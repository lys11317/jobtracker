# 求职管理器 Microsoft Edge 插件

版本：1.4.6

## 安装

1. 打开 Microsoft Edge。
2. 进入 `edge://extensions/`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择这个文件夹：`outputs/job-tracker-extension`。

## 使用

1. 打开招聘职位页或投递状态页。
2. 点击 Microsoft Edge 工具栏里的“求职管理器”插件图标。
3. 插件会读取当前页面，预填公司、职位、链接、渠道等信息。
4. 如果个人中心页面里识别到多条投递记录，可以先点选候选记录。
5. 手动确认或修改后点击“保存记录”。
6. 点击“管理器”查看、筛选、编辑、删除或导出 CSV。

## 数据

记录保存在 Microsoft Edge 插件本地存储 `chrome.storage.local` 中。卸载插件前请先导出 CSV。
